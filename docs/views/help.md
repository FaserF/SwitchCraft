# Help & Addon Manager

This view serves as both a resource center and a management interface for SwitchCraft's modular architecture.

## Addon Manager
SwitchCraft uses an "Addon" system to keep the core lightweight while allowing powerful optional features.

### Available Addons
*   **Advanced Core**: *Essential.* Enables Universal Analyzer, Brute Force detection, and advanced UI controls.
*   **AI Integration**: Enables the Helper view and AI context features.
*   **Winget Integration**: Enables Winget search and manifest features.

### Actions
*   **Install/Update**: Click the cloud download icon to install or update an addon from the official repository.
*   **Remove**: Uninstall an addon if it's causing issues or no longer needed.
*   **Status**: Indicates version and load status (Green = Active).

## Resources & Support
*   **Documentation**: Links to this documentation site.
*   **GitHub Issues**: Direct link to report bugs.
*   **Check for Updates**: Checks the main application version.

## Debugging
*   **View Logs**: Opens the real-time application log. Useful for troubleshooting errors or seeing what the Analyzer is doing under the hood.
*   **Open App Data**: Opens the `%AppData%\SwitchCraft` folder where logs and configs are stored.
